### run local
* go to /backend, run 
```
    $ docker build -t backend .      
    $ docker run -d -p 8080:8080 backend
```

* go to /frontend, run 
```
    $ docker build -t backend .      
    $ docker run -d -p 3000:3000 backend
```

* then visit http://localhost:3000
